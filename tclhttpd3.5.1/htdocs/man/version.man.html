NAME

httpd::version - Version information

TABLE OF CONTENTS

    TABLE OF CONTENTS
    SYNOPSIS
    DESCRIPTION
    Public API
    SEE ALSO
    KEYWORDS
    COPYRIGHT

SYNOPSIS

package require httpd::version ?3.4?

Httpd_Version

DESCRIPTION

The package httpd::version provides the version of the overall system.

Public API

Httpd_Version
Sets the version of the whole system into the global array Httpd provides by the package httpd. The changed key is version.
